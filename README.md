# ze_delivery

-> Criar conta
 - Campo email
 - Campo senha
 - Localização
 - Confirmar
 - Salvar informação em .txt ("Banco de dados local")
-> Fazer login
 - Usar login e senha cadastrado
 - Caso não seja, recusar e mostrar "Usuário não cadastrado"
-> Fazer pedido
 - Criar lista de objects a serem comercializados
 - Exibir texto com opções
 - Usuário faz pedido
-> Escolher forma de pagamento
 - Dinheiro
 - Cartão Crédito: número do cartão, cpf, data de validade, nome do titular
 - Cartão Débito: número do cartão, cpf, data de validade, nome do titular
-> Finalizar compra
 - Exibir informações de pedido, forma de pagamento e localização
-> Finalizar recebimento
 - Confirmar recebimento


OBS: projeto baseado em interação do usuário com linha de comando.
